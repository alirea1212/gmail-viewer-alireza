# GestureVerse AI

GestureVerse AI is a real-time hand tracking and gesture recognition platform powered by artificial intelligence. The system uses a device camera to detect and track hand landmarks point by point, then analyzes hand movements and gestures to convert them into emojis and visual reactions instantly.

The project is designed to work on both desktop computers and mobile devices through a modern web interface. Users can choose their device type at startup and interact with the system using natural hand gestures.

Key Features:

* Real-time hand tracking with landmark detection
* AI-powered gesture recognition
* Automatic emoji translation from hand signs
* Mobile and desktop support
* Smooth and responsive user interface
* Animated visual effects and graphics
* Large emoji library support
* Low-latency performance
* Modern AI-inspired dashboard
